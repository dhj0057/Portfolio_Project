# 0/1 LOGIC: Integrated Developer Identity System

> **"Zero Sentiment, Absolute Logic."** > 감정을 배제한 데이터 중심 분석과 정확한 로직을 지향하는 개발자 도형준의 통합 포트폴리오 시스템입니다.

---

## 1. 프로젝트 개요 (Project Overview)
* **프로젝트명:** 0/1 LOGIC (Integrated Developer Identity)
* **목적:** 1학년 1학기부터 졸업까지의 모든 기술적 자산(Artifacts)을 체계적으로 아카이브하고, 향후 AI 소프트웨어 엔지니어로서의 전문성을 증명하는 통합 대시보드 구축.
* 핵심 가치:  **Clarity:** 정보의 명확한 전달 및 위계 설정.
    * **Efficiency:** 불필요한 리소스를 제거한 미니멀리즘 아키텍처.
    * **Scalability:** 정적 페이지에서 SPA(React) 및 Full-stack 서비스로의 확장성 확보.

## 2. 기술 스택 (Tech Stack)
### Phase 1: Current (1-1)
* **Frontend:** HTML5 (Semantic Markup), CSS3 (Flexbox/Grid, 3D Transform), JavaScript (ES6+)
* **Version Control:** Git, GitHub
* **Deployment:** GitHub Pages

### Phase 2: Future Roadmap (1-2 ~ 2-2)
* **Library:** React.js (Component-based UI)
* **Backend:** Java Spring Boot
* **Database:** MariaDB / MySQL
* **Infrastructure:** AWS (EC2, S3)

## 3. 핵심 기능 요구사항 (Core Functional Requirements)
1.  **Identity Dashboard:** 사용자의 현재 기술 인벤토리 및 학습 상태(Linux Master 등)를 시각적으로 제시.
2.  **Project Archive:** 구현된 알고리즘(C#) 및 영상처리(Python) 프로젝트를 기술 명세와 함께 전시.
3.  **3D Logic Visualizer:** CSS 3D Transform을 활용한 입체적 시각화를 통해 공학적 인터랙션 제공.
4.  **Integrated Navigation:** 모든 섹션 및 외부 자산(GitHub)으로의 신속한 접근 허용.
